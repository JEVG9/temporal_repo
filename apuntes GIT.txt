
GIT VS GITHUB

Hay que tener en cuenta que GIT y GITHUB son completamente
diferentes, GIT permite el control de versiones y GITHUB es una 
herramienta/pagina/app que con el uso de git ayuda a mantener
informacion de proyectos,info personal, o lo que desees.

GIT

Pagina principal: https://git-scm.com

Esta herramienta permite tener un repositorio y control de
versiones de proyectos/apps/etc, para evitar perdida 
de datos, errores, misscomunication, etc.

GIT permite tener varios caminos para un proyecto y diversas 
funciones que permite un control temporal del proyecto tal que
una fotografia.

La mejor manera (para saber todo) es manejando GIT desde la
terminal.

Comandos principales a tener en cuenta:

    - ls: muestra todo lo accesible en el directorio 
    actual
    - cd: asignas de manera cd {file/dir} para posicionarte
    en ese lugar ( usando .. retornas)
    - pwd: indica el path donde estas
    - git --help: te muestra las opciones principales

Configuracion local en mi PC:

    - git config --global user.name "{username}"
    - git config --global user.email "{email}"
    estos dos comandos le indican a git quien estara usando 
    el sistema local aunque sea global(solo mi pc)

Crear repo:

    - git init: nos situamos en el directorio que trabajaremos
    y colocamos el comando, entonces git crea una carpeta 
    oculta que tiene toda la info de git y en la cual se genera
    un master

    - git branch -m {name}: con este comando renombramos la
    rama principal


Usando GIT

En GIT se trabaja por ramas, cada version es como una foto
de cada cambio que se le hace al codigo.

Para iniciar a trabajar lo principal es subir los ficheros,
en este caso por ejemplo estamos en la carpeta GIT luego tenemos
este mismo txt hacemos lo siguiente: git add "{fichero}"

    -git add "file": agrega un fichero
    -git status: te muestra el estado del branch
    -git commit: agrega una actualizacion, siempre pide comentarios
    para comentar de manera simple agregamos -m "{text}"
    -git checkout "{fichero}": retorna al anterior foto de ese fichero
    este comando tiene mas funciones ( REVISAR INFO )
    -git reset:indica que se desharan cambios en una area como tal
    este comando tiene mas funciones ( REVISAR INFO)
    -git log: este comando muestra todas los commits
